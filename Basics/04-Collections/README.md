# 04. Collections 🐍

## 📖 Overview

Collections are data structures used to store multiple values in a single variable. Python provides four built-in collection types: Lists, Tuples, Sets, and Dictionaries.

---

## 🎯 Learning Objectives

After completing this topic I should be able to:

- Understand Python collections
- Create and manipulate Lists
- Work with Tuples
- Use Sets for unique values
- Store key-value pairs using Dictionaries
- Apply common collection methods

## 💻 Example

```python
fruits = ["Apple", "Banana", "Mango"]

print(fruits)

student = {
    "name": "Anjani",
    "age": 22
}

print(student["name"])
```

---

## 🔑 Key Takeaways

- Lists are ordered and mutable.
- Tuples are ordered and immutable.
- Sets store unique values.
- Dictionaries store data as key-value pairs.
- Collections make data management efficient.
