# 06. Loops 🐍

## 📖 Overview

Loops are used to execute a block of code repeatedly without writing the same code multiple times. Python mainly provides **for** loops and **while** loops for iteration.

---

## 🎯 Learning Objectives

After completing this topic I should be able to:

- Understand why loops are used
- Write for loops
- Write while loops
- Use the range() function
- Create nested loops
- Control loops using break, continue, and pass

---

## 📚 Topics Covered

- for Loop
- while Loop
- range() Function
- Nested Loops
- break Statement
- continue Statement
- pass Statement
- Pattern Printing

---

## 💻 Example

```python
for i in range(1, 6):
    print(i)
```

---

## 🔑 Key Takeaways

- Loops reduce repetitive code.
- `for` loops are best when the number of iterations is known.
- `while` loops continue until a condition becomes false.
- `break` exits a loop.
- `continue` skips the current iteration.
- `pass` acts as a placeholder.
