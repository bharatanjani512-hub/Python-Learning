# 01. Variables and Data Types 🐍

## 📖 Overview

Variables are used to store information in memory.

Python automatically detects the data type of a variable, making it a dynamically typed language.

---

## 🎯 Learning Objectives

After completing this topic I should be able to:

- Create variables
- Understand different data types
- Perform type casting
- Take user input
- Print formatted output

---

## Topics Covered

- Variables
- Data Types
- Integers
- Floats
- Strings
- Boolean
- Type Casting

---

## Example

```python
name = "Anjani"

age = 22

print(name)

print(age)
```

---

## Key Takeaways

- Variables store data.
- Python automatically detects data types.
- Variable names should be meaningful.
- Every variable has a type.
