# Operators Notes

## What are Operators?

Operators are special symbols used to perform operations on values and variables.

Example:

```python
a = 10
b = 5

print(a + b)
```

---

## Types of Operators

### Arithmetic

Used for mathematical calculations.

```
+
-
*
/
/
//
%
**
```

---

### Assignment

Assign values.

```
=
+=
-=
*=
/=
```

---

### Comparison

Return True or False.

```
==
!=
>
<
>=
<=
```

---

### Logical

```
and
or
not
```

---

### Identity

```
is
is not
```

---

### Membership

```
in
not in
```

---

### Bitwise

```
&
|
^
~
<<
>>
```

---

## Key Takeaways

- Operators perform actions on data.
- Comparison operators return Boolean values.
- Logical operators combine conditions.
- Identity checks object identity.
- Membership checks presence inside collections.
