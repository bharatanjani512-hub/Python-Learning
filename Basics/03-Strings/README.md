# 03. Strings 🐍

## 📖 Overview

Strings are sequences of characters used to store text in Python.

---

## 🎯 Learning Objectives

After completing this topic I should be able to:

- Create strings
- Access characters
- Slice strings
- Use string methods
- Format strings

---

## Topics Covered

- Creating Strings
- Indexing
- Slicing
- String Methods
- String Formatting
- Escape Characters

---

## Example

```python
name = "Anjani"

print(name.upper())
print(name.lower())
```
---

## Key Takeaways

- Strings are immutable.
- Strings support indexing and slicing.
- Python provides many built-in methods.
