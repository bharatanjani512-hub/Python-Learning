# 05. Conditional Statements 🐍

## 📖 Overview

Conditional statements allow a program to make decisions based on conditions.

---

## 🎯 Learning Objectives

After completing this topic I should be able to:

- Use if statements
- Work with if-else
- Use nested if
- Write elif conditions
- Build decision-making programs

---

## 📚 Topics Covered

- if
- if-else
- elif
- Nested if
- Short-hand if
- Logical Operators

---

## 💻 Example

```python
age = 20

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

---

## 🔑 Key Takeaways

- Conditions return True or False.
- Python uses indentation.
- elif checks multiple conditions.
