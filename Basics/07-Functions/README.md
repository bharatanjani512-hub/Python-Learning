# 07. Functions 🐍

## 📖 Overview

Functions are reusable blocks of code designed to perform a specific task. They help organize programs, reduce repetition, and improve code readability.

---

## 🎯 Learning Objectives

After completing this topic I should be able to:

- Understand the purpose of functions
- Create user-defined functions
- Use built-in functions
- Pass parameters and arguments
- Return values from functions
- Write lambda (anonymous) functions

---

## 📚 Topics Covered

- Built-in Functions
- User-defined Functions
- Function Parameters
- Function Arguments
- Return Statement
- Lambda Functions

---

## 💻 Example

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("Anjani"))
```

---

## 🔑 Key Takeaways

- Functions make code reusable.
- Parameters receive values from arguments.
- The `return` statement sends values back.
- Lambda functions are useful for short operations.
